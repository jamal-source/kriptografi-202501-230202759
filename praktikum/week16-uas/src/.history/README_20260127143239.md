# KoinAllIn ERC-20 Token Project

This project creates a TinyCoin (TNC) ERC-20 token using Python, Brownie, Web3.py, and Ganache.

## Prerequisites

- Python 3.8+
- Node.js and npm
- Microsoft Visual C++ 14.0 or greater (for Windows, install from https://visualstudio.microsoft.com/visual-cpp-build-tools/)

## Installation

1. Install Python dependencies:
   ```bash
   pip install eth-brownie web3
   ```

   If you encounter build errors, try installing in a virtual environment:
   ```bash
   python -m venv venv
   venv\Scripts\activate  
   pip install eth-brownie web3
   ```

2. Install Ganache:
   ```bash
   npm install -g ganache
   ```

3. Install Brownie dependencies:
   ```bash
   brownie pm install
   ```

## Usage

1. Start Ganache (local blockchain):
   ```bash
   ganache
   ```

2. Deploy the KoinAllIn contract:
   ```bash
   brownie run skrip/deploy_patched.py --network mainnet
   ```

3. Interact with the contract (transfer tokens and check balances):
   ```bash
   brownie run skrip/interaksi.py --network mainnet
   ```

## Contract Details

- **Name**: KoinAllIn
- **Symbol**: AIC
- **Initial Supply**: 1,000,000 AIC (deployed to the first account)
- **Decimals**: 18 (standard ERC-20)
- **Features**: Burning, Staking

## Scripts

- `deploy_patched.py`: Deploys the KoinAllIn contract with initial supply and MetaMask integration.
- `interaksi.py`: Interacts with the contract (transfer tokens and check balances).
